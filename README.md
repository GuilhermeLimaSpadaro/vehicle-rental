VehicleManager

VehicleManager é um sistema em Java para cadastro e gerenciamento de veículos e clientes, desenvolvido como projeto de estudo de programação orientada a objetos, arquitetura em camadas e boas práticas de design de software.

📝 Objetivo

Aprender e aplicar conceitos de POO, como herança, interfaces, polimorfismo e encapsulamento.

Organizar código usando arquitetura em camadas: entities, services, interfaces, exceptions e main.

Desenvolver um sistema funcional de cadastro e gerenciamento de veículos (Carro, Moto, Caminhão) e seus proprietários.

📂 Estrutura do Projeto

entities: classes que representam os objetos do sistema (Car, MotorCycle, Truck, Proprietário, Motor).

interfaces: contratos que definem métodos essenciais para veículos e operações do sistema.

services: lógica de negócio, como adicionar, remover, listar e buscar veículos.

exceptions: classes para tratamento de erros e validações.

main: classe principal com menu interativo no console.

🚗 Funcionalidades

Cadastro de veículos e proprietários.

Validações de dados para evitar inconsistências.

Listagem de todos os veículos cadastrados.

Busca de veículos por modelo.

Cálculo de impostos (IPVA) de forma automatizada.

Diferentes tipos de veículos com atributos específicos.

Uso de equals e hashCode para garantir unicidade de registros.

Menu em console para interação simples com o usuário.
